An AI-powered financial advisor system built with Spring Boot and React, implementing a multi-agent architecture with LangChain4j and Groq. The runtime currently uses **12 LLM agent roles**: 5 core specialist agents (User Profile, Market Analysis, Web Search, Fintwit, Security), 3 orchestration/governance agents (Orchestrator, Planner, Evaluator), and 4 ticker-resolution consensus agents (Planner, Selector, Evaluator, Auditor). Key technical capabilities include plan-execute-evaluate retry loops, independent auditing for symbol selection, real-time WebSocket streaming of reasoning/tool activity, live market integration via Finnhub, Google OAuth + JWT authentication, portfolio management with automatic price refresh, and strong safety controls. Built with Java 21, Spring Boot, LangChain4j, React, PostgreSQL, and Railway deployment.
